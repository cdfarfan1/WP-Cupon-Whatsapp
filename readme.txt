Plugin para programa de fidelización y canje de cupones por WhatsApp integrado con WooCommerce. Versión limpia y optimizada.

== Description ==

WP Cupón WhatsApp es un plugin que permite implementar un sistema de programa de lealtad donde los clientes pueden obtener cupones y canjearlos a través de WhatsApp.

= Características principales: =

* Sistema de cupones de fidelización
* Integración con WhatsApp para canjes
* Compatible con WooCommerce
* Panel de administración intuitivo
* Base de datos optimizada
* Código limpio y eficiente

= Requisitos: =

* WordPress 5.0 o superior
* PHP 7.4 o superior
* WooCommerce 6.0 o superior

== Installation ==

1. Sube el plugin a la carpeta `/wp-content/plugins/`
2. Activa el plugin desde el menú 'Plugins' en WordPress
3. Ve a 'WP Cupón WhatsApp' en el menú de administración
4. Configura las opciones según tus necesidades
