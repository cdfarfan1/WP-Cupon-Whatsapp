Plugin para programa de fidelización y canje de cupones por WhatsApp integrado con WooCommerce. Versión corregida con problemas de headers resueltos, formularios mejorados y optimizaciones de rendimiento.

== Description ==

Este plugin permite implementar un sistema de programa de lealtad donde los clientes pueden obtener cupones y canjearlos a través de WhatsApp. Los comercios e instituciones pueden gestionar sus cupones y ver estadísticas de uso.

Principales funcionalidades:
* Gestión de Comercios e Instituciones (Custom Post Types).
* Creación de cupones de lealtad y cupones públicos.
* Proceso de solicitud de adhesión para nuevos comercios/instituciones.
* Flujo de canje de cupones mediante WhatsApp.
* Confirmación de canje por parte del negocio.
* Panel de "Mis Canjes" para clientes.
* Integración con pedidos de WooCommerce para seguimiento.
* Roles de usuario personalizados para Comercios e Instituciones.

== Installation ==

1. Sube la carpeta `wp-cupon-whatsapp` al directorio `/wp-content/plugins/`.
2. Activa el plugin a través del menú 'Plugins' en WordPress.
3. Ve a "WP Canje Cupon Whatsapp > Ajustes" para configurar el plugin.
