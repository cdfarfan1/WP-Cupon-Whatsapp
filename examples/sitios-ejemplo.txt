# Archivo de ejemplo para despliegue masivo de WP Cupón WhatsApp
# Formato: URL [perfil] [optimización]
# 
# PERFILES DISPONIBLES:
# - minimal: Configuración básica para sitios pequeños
# - standard: Configuración estándar (recomendado)
# - enterprise: Configuración empresarial para alto tráfico
# - development: Configuración para entornos de desarrollo
#
# OPTIMIZACIONES DISPONIBLES:
# - none: Sin optimizaciones
# - basic: Optimizaciones básicas (minificación, lazy loading)
# - standard: Optimizaciones estándar (recomendado)
# - aggressive: Optimizaciones agresivas (máximo rendimiento)
#
# EJEMPLOS DE USO:

# === SITIOS DE PRODUCCIÓN ===

# Sitios corporativos con alto tráfico
https://empresa1.com enterprise aggressive
https://empresa2.com enterprise aggressive
https://tienda-online.com enterprise standard

# Sitios estándar de clientes
https://cliente1.com standard standard
https://cliente2.com standard standard
https://blog-personal.com standard basic
https://pequeña-empresa.com standard basic

# Sitios básicos o con pocos recursos
https://sitio-basico.com minimal basic

# === NOTAS IMPORTANTES ===
#
# 1. Cada línea representa un sitio a configurar
# 2. Las líneas que empiezan con # son comentarios
# 3. Si no se especifica perfil, se usa 'standard'
# 4. Si no se especifica optimización, se usa 'standard'
# 5. Los sitios deben ser accesibles y tener WordPress instalado
# 6. Se recomienda hacer una copia de seguridad antes del despliegue
# 7. Usar modo --test primero para verificar la configuración
#
# COMANDOS DE EJEMPLO:
#
# Linux/Mac:
# ./scripts/mass-deployment.sh examples/sitios-ejemplo.txt
# ./scripts/mass-deployment.sh --test examples/sitios-ejemplo.txt
# ./scripts/mass-deployment.sh -p enterprise -o aggressive examples/sitios-ejemplo.txt
#
# Windows:
# .\scripts\mass-deployment.ps1 -SitesFile examples\sitios-ejemplo.txt
# .\scripts\mass-deployment.ps1 -SitesFile examples\sitios-ejemplo.txt -TestMode
# .\scripts\mass-deployment.ps1 -SitesFile examples\sitios-ejemplo.txt -Profile enterprise -Optimization aggressive
